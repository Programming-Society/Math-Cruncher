#FileSpecifications
Describes the files that the program will create and use.

// userLog.bin
// stores username/password pairs in a standard format.
// sample format listed below
<USER>
userName="John Doe"
password="p@$sW0"
</USER>
<USER>
userName="Jane Doe"
password="WEL:KRJ"
</USER>

// scores.bin
// stores the score records for each user in a standard format.
// sample formated listed below
<USER>
userName="John Doe"
13
24
48
</USER>
<USER>
userName="Jane Doe"
35
23
10
0
</USER>
